Voyage+ is a companion to Voyage Theme. It provides theme-independent features such as shortcodes, action hooks, etc.

== Description ==

Voyage was created to incorporate theme-independent features:

* Ad Unit Shortcode: `[ad]`
* Reponsive Grid Columns shortcode that strickly follows grid system: `[column]`
* action hooks to wp_head() and wp_footer()

Support is provided at <a href="http://www.voyagebc.com/support">support forums</a>.

== Installation ==

1. Unzip the `voyageplus.zip`.
1. Upload the `voyageplus` folder to your `/wp-content/plugins` directory.
1. Activate *Voyage Plus* in Plugin section
